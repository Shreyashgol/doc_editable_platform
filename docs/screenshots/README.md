# Screenshots

Place PNG captures here with these exact filenames so they render in the root `README.md`:

| File | What to capture |
|------|-----------------|
| `dashboard.png` | Documents dashboard with a few rows and status badges |
| `canvas.png` | A document's canvas — symbols drawn, one selected, property editor panel open |
| `graph.png` | The relationship graph viewer for a processed document |
| `search.png` | The semantic search page with a query and results |
| `upload.png` | The upload page |
| `api-docs.png` | `https://<your-api>/docs` (Swagger UI) |

Tips
- Use a consistent theme (or show off both — e.g. dashboard in dark, canvas in light).
- ~1600px wide, trimmed to the content; keep each file under ~500 KB.
- macOS: `Cmd-Shift-4` then drag; or capture the browser window with `Cmd-Shift-5`.
